English:

============================================================
SECTION 1: SYSTEM SETUP (FROM ZERO)
============================================================
Step 1: Install prerequisites
English:
Install PHP 8.2+, Composer, Node.js + npm, MySQL, and Git.

Hindi:
PHP 8.2+, Composer, Node.js + npm, MySQL aur Git install karein.

Step 2: Project open karein
English:
Open project folder in terminal: C:\xampp\htdocs\meerahr

Hindi:
Terminal me project folder open karein: C:\xampp\htdocs\meerahr

Step 3: Environment file banayein
English:
Copy .env.example to .env and update database values.

Hindi:
.env.example ko copy karke .env banayein aur database values update karein.

Suggested DB values:
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=School_DB_all
DB_USERNAME=root
DB_PASSWORD=

Step 4: App key generate karein
English:
Run: php artisan key:generate

Hindi:
Command chalayein: php artisan key:generate

Step 5: Dependencies install karein
English:
Run: composer install
Run: npm install

Hindi:
Command chalayein: composer install
Command chalayein: npm install

Step 6: Database migrate + seed
English:
Run: php artisan migrate --seed

Hindi:
Command chalayein: php artisan migrate --seed

Step 7: Storage link banayein
English:
Run: php artisan storage:link

Hindi:
Command chalayein: php artisan storage:link

Step 8: Frontend build karein
English:
Run: npm run build

Hindi:
Command chalayein: npm run build

Step 9: Application start karein
English:
Run: php artisan serve
Open: http://127.0.0.1:8000

Hindi:
Command chalayein: php artisan serve
Open karein: http://127.0.0.1:8000

SECTION 16: TROUBLESHOOTING QUICK HELP
============================================================
Issue: Page blank or Blade error
English: Run php artisan view:clear and php artisan view:cache
Hindi: Command chalayein php artisan view:clear aur php artisan view:cache

Issue: CSS/JS not updating
English: Run npm run build and hard refresh browser
Hindi: npm run build karein aur browser hard refresh karein

Issue: Images/files not showing
English: Run php artisan storage:link and check file permissions
Hindi: php artisan storage:link chalayein aur file permissions check karein

Issue: Login not working
English: Verify credentials and check users table active flag
Hindi: Credentials verify karein aur users table me active flag check karein
