==================================================
DAY 0 (PRE-TRAINING TECH SETUP)
==================================================
Goal:
Environment ready + login working.

EN Steps:
1. Install PHP 8.2+, Composer, Node.js, npm, MySQL.
2. Open project folder: C:\xampp\htdocs\meerahr
3. Create .env from .env.example
4. Configure database:
   DB_CONNECTION=mysql
   DB_HOST=127.0.0.1
   DB_PORT=3306
   DB_DATABASE=School_DB_all
   DB_USERNAME=root
   DB_PASSWORD=
5. Run:
   composer install
   npm install
   php artisan key:generate
   php artisan migrate --seed
   php artisan storage:link
   npm run build
   php artisan serve
6. Open app: http://127.0.0.1:8000

HI Steps:
1. PHP 8.2+, Composer, Node.js, npm, MySQL install karein.
2. Project folder open karein: C:\xampp\htdocs\meerahr
3. .env.example se .env banayein.
4. Database values set karein (upar wali).
5. Commands chalayein:
   composer install
   npm install
   php artisan key:generate
   php artisan migrate --seed
   php artisan storage:link
   npm run build
   php artisan serve
6. App open karein: http://127.0.0.1:8000

Validation Check:
- Login page open ho
- Super admin login work kare
